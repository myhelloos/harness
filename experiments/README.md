# Experiments

Each experiment should be small enough to rerun.

Use this structure:

```text
experiments/YYYY-MM-DD-short-name/
├── README.md
├── setup.md
├── run.md
├── results.md
└── artifacts/
```

## Experiment Template

Include:

- Question.
- Harness or framework tested.
- Model and version.
- Task.
- Budget.
- Commands.
- Raw output location.
- Result.
- Failure notes.
- Next decision.
